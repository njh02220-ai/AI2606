# AICE CBT 전체 코드 ver1.1

이 압축 파일에는 실행 목적에 따라 두 가지 버전이 들어 있습니다.

## 1. 화면을 바로 확인하는 단일 파일

`AICE_CBT_4단계_캔버스.html`을 더블클릭해 웹브라우저로 열면 됩니다.

- CSS와 JavaScript가 HTML 내부에 포함되어 있습니다.
- 홈, 주제별 학습, 모의고사, 결과, 오답노트를 한 파일에서 확인할 수 있습니다.
- 파일 연결 누락으로 빈 화면이 생기는 문제를 피한 버전입니다.

## 2. 수정·GitHub 업로드용 전체 프로젝트

`aice-cbt-study` 폴더에는 역할별로 분리한 전체 코드가 들어 있습니다.

- HTML: 5개 화면
- CSS: 공통 디자인
- JavaScript: 문제 데이터, 저장, 학습, 모의고사, 결과, 오답노트 기능
- 테스트: 문제 데이터와 저장 분리 확인
- README: 실행 방법과 파일별 역할

분리형 프로젝트는 ES 모듈을 사용하므로 `index.html`을 직접 더블클릭하기보다 폴더에서 다음 명령을 실행하는 것을 권장합니다.

```bash
cd aice-cbt-study
python3 -m http.server 8000
```

그다음 웹브라우저에서 `http://localhost:8000`을 엽니다.

자동 검사는 아래 명령으로 실행할 수 있습니다.

```bash
cd aice-cbt-study
npm test
```

## 현재 범위

- 직접 제작한 학습용 예시 문제 5개
- 브라우저 내부 기록 저장
- 학습·모의고사·오답·재풀이 기록 분리
- 모의고사 타이머와 제출 확인
- 자동 채점, 문항별 해설, 오답 복습 메모

회원가입, 서버, 데이터베이스, 클라우드 동기화, Python 코드 실행은 포함하지 않습니다.
