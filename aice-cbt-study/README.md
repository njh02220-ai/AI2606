# AICE 대비 AI 학습 CBT

AI 기초·데이터 이해·생성형 AI의 직접 제작 학습용 문제를 풀 수 있는 개인용 정적 웹앱입니다. 주제별 학습, 10분 모의고사, 결과·해설, 오답 복습을 제공하며 서버 없이 브라우저에 기록을 저장합니다.

## 실행

프로젝트 폴더에서 아래 명령을 실행한 뒤 `http://localhost:8000`을 여세요.

```bash
python3 -m http.server 8000
```

ES 모듈을 사용하므로 `index.html` 파일을 직접 더블클릭하기보다 간단한 로컬 서버에서 실행하는 것을 권장합니다.

기본 데이터·채점·저장 분리 검사는 아래 명령으로 실행할 수 있습니다.

```bash
npm test
```

## 화면과 파일

| 수정 대상 | 담당 파일 |
|---|---|
| 홈·학습 현황 | `index.html`, `js/home.js` |

## 문제 데이터 최소 형식

```js
{
  id: "BAS-006",
  topic: "생성형 AI",
  stem: "문제 본문",
  choices: ["보기 1", "보기 2"],
  answer: [1],
  explanation: "정답 근거와 핵심 개념"
}
```

`id`는 중복되면 안 되며, 현재 1차 버전은 객관식 단일 정답 문제만 지원합니다. 문제 데이터에 필수 항목 누락이나 잘못된 정답 번호가 있으면 화면 대신 오류 안내를 표시합니다.

## 저장 안내

- 학습 기록, 모의고사 결과, 오답노트, 재풀이 상태는 서로 구분해 저장합니다.
- 저장 기록은 같은 브라우저에서만 유지됩니다.
- 브라우저 데이터를 삭제하거나 다른 기기로 바꾸면 기록이 사라질 수 있습니다.
- 기존 초기 버전의 혼합 기록은 통계에 섞지 않고 보관용 기록으로 이동합니다.

## 이용 범위

포함된 문제는 직접 제작한 학습 예시입니다. 실제 시험 기출문제나 공식 해설은 이용 허가와 재배포 범위를 확인하기 전에는 등록하지 마세요.
